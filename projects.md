### NeuraTube - AI-Powered Web Extension

**Tech Stack:** Built using Gemini 2.5 Flash API, Chrome Extension API (Manifest V3), and Vercel for backend proxying.

**Implementation:** Developed a seamless UI using JavaScript, HTML, and CSS to deliver real-time YouTube video insights.

**Impact:** Integrated AI-driven automated summarization, reducing content review time for long-form videos by an estimated 60%.
